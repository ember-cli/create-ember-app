create-ember-app
==============================================================================

```sh
yarn create ember-app <name>
```

or if you have npm@6.1 or later:

```sh
npm init ember-app <name>
```

This is a small wrapper for [Ember CLI](https://github.com/ember-cli/ember-cli)
and specifically the `ember new` command.
